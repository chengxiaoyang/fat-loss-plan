  • 力量：60–70分钟

周二：有氧+核心

  • 爬坡走 30分钟
  • 速度 5.5 km/h
  • 坡度 8–10

周三：腿+核心+有氧

  • 力量：60–70分钟

从恢复 → 燃脂 → 腹肌清晰

1–4周｜恢复期

重量：50–60%

重点：保护关节、找回感觉

5–8周｜燃脂期

重量+10–15%

饮食计划

减脂成功，7分靠吃，执行即见效

一、每日目标（核心数据）

热量节奏：

每日热量赤字 500–700 kcal

目标：2000 kcal / 天

宏量营养：

蛋白质：150–170 g

脂肪：50–60 g

碳水：180–220 g

原则：蛋白质优先 → 碳水 → 控脂肪

二、每日饮食结构

3餐 + 1加餐（最稳、最容易坚持）

  • 早餐
  • 午餐
  • 训练前/后加餐
  • 晚餐

三、标准一天饮食模板（推荐）

早餐（≈450 kcal）

鸡蛋3个 + 燕麦50g + 无糖酸奶/牛奶

蛋白质 ≈ 35g

午餐（≈650 kcal）

鸡胸/牛肉 200g + 米饭100g + 蔬菜随便吃

蛋白质 45–50g

训练加餐（≈200 kcal）

蛋白粉1份 或 无糖酸奶+香蕉

晚餐（≈550 kcal）

鱼肉/鸡胸200g + 红薯150g + 大量蔬菜

不训练 → 碳水再减一点

四、汉堡王怎么吃

推荐：

1个牛肉汉堡 + 2个鸡腿

无糖可乐 / 水

不要：薯条、甜饮

热量：700–850 kcal

一周 2–3 次完全没问题

五、外卖/便利店指南

中餐：

烤鱼、牛肉饭、鸡胸饭、少油盖浇饭

规则：肉多、油少、饭半份

便利店：

2茶叶蛋 + 鸡胸肉 + 无糖酸奶 + 香蕉

≈400–500 kcal

六、减脂成功率最高的5条规则

  • 1. 蛋白质一定要够（≥150g）
  • 2. 晚上少油
  • 3. 饮料只喝无糖
  • 4. 拒绝隐藏热量（坚果、奶茶）
  • 5. 睡前不吃高碳水

关键技巧：

早餐正常、午餐正常、晚餐减碳水 → 减脂速度明显更快
